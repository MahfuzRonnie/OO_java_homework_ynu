J2SE 1.5 Documentation (API)
Write, compile and execute Hello World application and other examples in the textbook in the dos environment and Jcreator Open command prompt.

To edit source code:
Type notepad HelloWorld.java.
Type in the following:
    public class HelloWorld {
        public static void main (String[] args) {
            System.out.println("Hello World!");
        }
    }
Save the file.

To compile source code to byte code:
At command prompt, type javac HelloWorld.java.

To execute byte code:
At command prompt, type java HelloWorld
